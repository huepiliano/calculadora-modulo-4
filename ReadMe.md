Análisis del Proyecto – Calculadora en Consola

Este proyecto consiste en el desarrollo de una aplicación de consola en JavaScript que funciona como una calculadora básica. Permite realizar operaciones de suma, resta, multiplicación y división, además de guardar un historial de resultados.

El programa solicita el nombre del usuario al iniciar y luego muestra un menú con distintas opciones. Se utilizó un ciclo while para mantener la aplicación en funcionamiento hasta que el usuario decida salir.

Para organizar el código se utilizaron:

Variables para almacenar datos.

Funciones para las operaciones matemáticas.

Un objeto para estructurar parte del programa.

Un arreglo para guardar el historial.

Métodos como filter y map para mostrar solo los resultados pares.

También se implementó validación de datos para asegurar que el usuario ingrese números válidos y se controló el caso de división por cero.

En general, el proyecto permitió aplicar los contenidos del módulo 4, especialmente el uso de funciones, objetos, arreglos y estructuras de control en JavaScript.
